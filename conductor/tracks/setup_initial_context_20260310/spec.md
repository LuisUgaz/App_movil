# Especificación: Contexto inicial del proyecto y características

## Objetivo
Establecer la base técnica de la aplicación móvil y mostrar el listado de proyectos cargado desde un archivo JSON local.

## Requisitos
1. **Estructura del Proyecto:** Organizar el código en carpetas lógicas (src, components, data, screens).
2. **Datos Locales:** Crear un archivo JSON con datos de prueba de al menos 3 proyectos.
3. **Listado de Proyectos:** Implementar una pantalla que lea el JSON y muestre los proyectos en tarjetas visuales.
4. **Detalle de Proyecto:** Permitir navegar a una pantalla de detalle al seleccionar un proyecto.

## Criterios de Aceptación
- La aplicación muestra el listado de proyectos correctamente.
- Los datos se cargan dinámicamente desde el archivo JSON local.
- La navegación entre el listado y el detalle funciona sin errores.
- El código sigue las guías de estilo de TypeScript y React definidas.
